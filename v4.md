# Expense Tracker - Version 4

## Changes from v3

1. **Table view instead of tabs**
   - Replaced card-style expense items with a clean table layout
   - Main and Self expenses each have their own collapsible dropdown section
   - Table columns: Amount, Category, Date, Note, Actions (Edit / Delete)

2. **Month dropdowns sorted latest to oldest**
   - Months are now sorted in reverse chronological order (newest first)
   - Uses proper date parsing to ensure correct ordering

3. **Green success toast on expense added**
   - A green popup appears at the top of the screen for ~1.5 seconds when a new expense is successfully saved
   - Smoothly fades out after the delay
   - Only shown for new additions, not edits

4. **Add Expense button shows − when expanded**
   - Button label changes from `+ Add Expense` to `− Add Expense` when the form is open
   - Reverts to `+` when the form is closed or after saving

5. **Date field placeholder visible**
   - Date input now shows a greyed-out placeholder color when empty
   - Color changes to dark when a date is selected

6. **+ Salary button per month with remaining balance**
   - Each month header has a `+ Salary` button (turns `✏️ Salary` after set)
   - Clicking opens a modal to enter the month's salary
   - Once set, the summary block shows: Salary, Main Total, Self Total, and Remaining
   - Remaining is shown in green (within budget) or red (over budget)
   - Salary data persists via localStorage
