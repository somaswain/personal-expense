# Expense Tracker — Version 5

## Changes from v4

1. **Options icon instead of Edit/Delete buttons**
   - Replaced inline Edit + Del buttons with a single `⋯` options trigger per row
   - Clicking opens a floating context menu (Edit / Delete) positioned near the button
   - Menu closes on outside click — keeps the table clean and prevents overflow

2. **Category column removed from table**
   - Table now shows: Amount · Date · Note · Options
   - Leaner layout fits mobile screen without horizontal overflow

3. **"crafted by somaswain" tagline**
   - Displayed as a subtle muted line beneath the main heading

4. **Full CSS revamp — refined minimal aesthetic**
   - Font: DM Serif Display (headings) + DM Sans (body) via Google Fonts
   - Warm neutral palette (`#F7F6F3` background, off-white cards, warm borders)
   - CSS variables throughout for consistent theming
   - Bottom-sheet salary modal with backdrop blur
   - Animated floating options menu (pop-in keyframe)
   - Smooth chevron rotation on section expand/collapse
   - Compact 2-column form grid for faster entry
   - Label + input groups with uppercase micro-labels

5. **Export & Import**
   - **Export**: Download button in header — saves a `.json` backup file with all expenses + salaries and a timestamp
   - **Import**: Upload button in header — reads a backup `.json` and merges new entries (skips duplicates by ID)
   - Salaries from backup are merged (imported month values win)
   - Works entirely client-side — no server needed; safe for browser-data-clear scenarios
